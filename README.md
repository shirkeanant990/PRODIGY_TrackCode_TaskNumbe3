## PRODIGY_DataScience_TaskNumbe3
A Python based sentiment analysis project that process and analyze Twitter data using NLP techniques .It uses TextBlob to classify  sentiments  and  visualization of insights  with Matplotlib and Seaborn
---

## Objectives
- Perform data cleaning and preprocessing  
- Analyze text data using sentiment analysis  
- Classify sentiments into Positive, Negative, and Neutral  
- Visualize sentiment distribution and topic-wise analysis  

---

## Dataset
The dataset contains Twitter data with Null Heading columns added following columns:
- `id` – Unique identifier  
- `topic` – Topic or brand  
- `sentiment` – Original sentiment label  
- `text` – Tweet content  

---

---

## Technologies Used
 Python  
- Pandas  
- Matplotlib  
- Seaborn  
- TextBlob  

---

## Conclusion
This project demonstrates how sentiment analysis can be applied to social media data to extract meaningful insights. By cleaning and processing textual data, and applying TextBlob for sentiment classification, the project helps in understanding public opinion across different topics. The visualizations further enhance interpretability, making it easier to identify trends and patterns in user sentiments.
